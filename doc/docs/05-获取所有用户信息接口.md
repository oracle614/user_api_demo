# 05-获取所有用户信息接口
    
**简要描述：** 

获取所有用户姓名和密码(需要登录接口练习)

**请求URL：** 
` http://127.0.0.1:5000/api/user/getUserList/`
  
**请求方式：**
GET

**参数**
无

 **返回示例**
成功：

|ID    |用户名|密码                            |
|:-----|------|--------------------------------|
|1     |张三  |e10adc3949ba59abbe56e057f20f883e|
|2     |李四  |7235ce1a737d16c95a51e3f9efabaea5|
|3     |王五  |e10adc3949ba59abbe56e057f20f883e|

失败：

失败，尚未登录

**备注**
需要登录获取session或登录一次后携带cookies
